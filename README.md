# Smart-Web-Utility-Hub
Smart Web Utility Hub – A modern JavaScript web application featuring smart greetings, a live digital clock, dynamic color controls, an advanced calculator with history, and a responsive glassmorphism UI.

## Features

* Smart Greeting System based on current time
* Live Digital Clock
* Dynamic Color-Changing Button
* Click Counter
* Advanced Calculator
* Calculation History
* Responsive Design
* Glassmorphism UI
* Custom White Striped Triangle Background

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

├── index.html
├── style.css
├── script.js
└── assets/
